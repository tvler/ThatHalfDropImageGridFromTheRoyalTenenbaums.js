Setting an image URL

•String value

•Contains the image size within the element if there's no other property (Same thing as "background-size: contain")

•If src isn't stated, it attempts to inherit a URL from the element's background-image CSS property

<figure
  data-offset = '{
    "src" : "wes1.jpg"
  }
></figure>

Snap

• Boolean value

• Changes width of image to create a whole number of columns

•Only respects one min/max property

•If no min/max property is set, it creates 3 columns

<figure
  data-offset = '{
    "src"      : "wes1.jpg",
    "maxWidth" : 250,
    "snap"     : true
  }
></figure>
<figure
  data-offset = '{
    "src"       : "wes1.jpg",
    "minHeight" : 170,
    "snap"      : true
  }
></figure>
<figure
  data-offset = '{
    "src"  : "wes1.jpg",
    "snap" : true
  }
></figure>

Width and height

•[String | Double] positive value

•String values must end in “%” or “px”

•Percent values are relative to the element's same dimension (Same behavior as CSS background-size)

•If nothing is specified, its value is “auto”

•If both the width and height properties are set, the image aspect ratio changes

<figure
  data-offset = '{
    "src"   : "wes1.jpg",
    "width" : "33.33%"
  }
></figure>
<figure
  data-offset = '{
    "src"    : "wes1.jpg",
    "width"  : "100%",
    "height" : "100%"
  }
></figure>
<figure
  data-offset = '{
    "src"    : "wes1.jpg",
    "width"  : "100%",
    "height" : "33.33%"
  }
></figure>

Max width/height

•[String | Double] positive value

•String values must end in “%” or “px”

•Percent values are relative to the element's same dimension (Same behaivior as CSS background-size)

•If nothing is specified, its value is “none”

<figure
  data-offset = '{
    "src"      : "wes1.jpg",
    "width"    : "33.33%",
    "maxWidth" : 150
  }
></figure>
<figure
  data-offset = '{
    "src"       : "wes1.jpg",
    "width"     : 150,
    "minHeight" : "80%"
  }
></figure>
